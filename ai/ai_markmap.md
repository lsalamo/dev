# AI (ARTIFICIAL INTELLIGENCE)

## AI Code

### [Cursor (Free)](https://www.cursor.com/) es un editor de código mejorado con IA diseñado para aumentar la productividad al permitir a los desarrolladores interactuar con su base de código a través de IA conversacional y comandos de lenguaje natural. Cursor es una bifurcación de VS Code. Esto nos permite centrarnos en crear la mejor manera de codificar con IA y, al mismo tiempo, ofrecer una experiencia de edición de texto familiar.

### Anthropic

#### [Claude 3.5 Sonnet (Free)](https://www.anthropic.com/news/claude-3-5-sonnet) es el último modelo de IA en lenguaje natural presentado por Anthropic, una empresa fundada por Dario Amodei, anteriormente de OpenAI. Resolvió el 64% de los problemas, superando a Claude 3 Opus que resolvió el 38%.

### Google

#### [Gemini (Free)](https://gemini.google.com/app) puede ayudar con tareas de programación y desarrollo de software, incluida la generación de código, la depuración y la explicación del código.

### [OpenAI](https://openai.com/)

#### [OpenAI Codex](https://openai.com/index/openai-codex/) es la herramienta de código de IA más destacada disponible en la actualidad. Se basa en los LLM de OpenAI (GPT-3 y GPT-4) y está entrenado en miles de millones de líneas de código. La herramienta domina más de una docena de lenguajes de programación.

#### [GitHub Copilot](https://github.com/features/copilot) utiliza código disponible públicamente en los repositorios de GitHub para que los usuarios puedan acceder a grandes conjuntos de datos y desarrollar rápidamente código preciso. La herramienta detecta errores en el código y recomienda cambios.

- [Getting code suggestions in your IDE with GitHub Copilot](https://docs.github.com/en/copilot/using-github-copilot/getting-code-suggestions-in-your-ide-with-github-copilot)

#### [ChatGPT GPT-3/4 (Free)](https://openai.com/index/chatgpt/) spuede ampliar su conocimiento contextual y sonar muy humano. Tiene un alcance más amplio de respuestas e incluso puede generar código. Pero al igual que con cualquier otra herramienta de inteligencia artificial, el código ChatGPT puede incluir errores, ser difícil de mantener, incluir vulnerabilidades de seguridad, etc.

### Meta

#### [Meta AI](https://www.meta.ai/) está impulsada por el modelo [Llama 3](https://llama.meta.com/llama-downloads/) con 70 mil millones de parámetros.


#### [Microsoft - Visual Studio IntelliCode (Free)](https://visualstudio.microsoft.com/es/services/intellicode/) es una extensión del editor Visual Studio Code creado por Microsoft que proporciona experiencias de desarrollo asistidas por IA para mejorar la productividad de los desarrolladores.

- [Extension VS Code](https://marketplace.visualstudio.com/items?itemName=VisualStudioExptTeam.vscodeintellicode)

### JetBrains

#### [PyCharm](https://www.jetbrains.com/pycharm/) es una herramienta de finalización de código con IA de JetBrains que proporciona a los desarrolladores capacidades de finalización de código inteligente

### Amazon

#### [Amazon Q (Free)](https://aws.amazon.com/es/q/developer/) es una herramienta desarrollada por Amazon. Ofrece sugerencias de código basadas en IA en tiempo real e identifica posibles coincidencias de código fuente abierto para facilitar su revisión.

- [Extension VS Code](https://marketplace.visualstudio.com/items?itemName=AmazonWebServices.amazon-q-vscode)
- [Extension JetBrains](https://plugins.jetbrains.com/plugin/24267-amazon-q/)

### Snyk

#### [DeepCode (Free)](https://snyk.io/platform/deepcode-ai/) es una herramienta de análisis de código de IA basada en la nube que escanea automáticamente la base de código de un proyecto e identifica posibles errores y vulnerabilidades.

### Open Source

#### [Tabnine (Free)](https://www.tabnine.com/) es una herramienta de finalización de código de IA que utiliza deep learning algorithms para proporcionar al usuario capacidades de finalización de código inteligente. Esta herramienta es de código abierto y la utilizan empresas tecnológicas líderes como Facebook y Google.

- [Extension VS Code](https://www.tabnine.com/install/visual-studio/)
- [Extension PyCharm](https://www.tabnine.com/install/pycharm/)

#### [Cody (Free)](https://sourcegraph.com/cody) es la herramienta de inteligencia artificial de Sourcegraph, es un asistente de codificación integral. Entiende todo su código base, responde consultas y escribe código. Cody proporciona explicaciones detalladas del código, localiza componentes específicos e identifica problemas potenciales con soluciones sugeridas.

- [Extension VS Code](https://marketplace.visualstudio.com/items?itemName=sourcegraph.cody-ai)

#### [CodiumAI (Free)](https://www.codium.ai/) es un moderno asistente de redacción de pruebas de IA que garantiza que su código funcione según lo previsto.

- [Extension VS Code](https://marketplace.visualstudio.com/items?itemName=Codium.codium)
- [Extension PyCharm](https://plugins.jetbrains.com/plugin/21206-codiumate--code-test-and-review-with-confidence--by-codiumai)

#### [Zapier Central (Free)](https://central.zapier.com/)

### Regex

#### [AutoRegex](https://www.autoregex.xyz/) traduce lenguaje natural a expresiones regulares

### Shell

#### [Warp](https://www.warp.dev/warp-ai) es una terminal moderna basada en Rust con IA integrada. Escriba "#" en su línea de comando y comience a describir el comando que desea ejecutar usando lenguaje natural. Warp cargará sugerencias de comandos de IA a medida que escribes.
